## A Era da Automação Extrema Documental
O desenvolvimento '0 a 1' de um SaaS inteiro dedicado ao fluxo burocrático de licitações. Um projeto monumental para transformar semanas exaustivas na escrita de ETPs e DFDs em documentos validados construídos em menos de 'cinco minutos'.

## TALL Stack no Backend
A aplicação principal gira via Laravel 11. Lidamos silenciosamente e agilmente com dezenas de instâncias com as famosas Laravel Queues & Jobs (Workers assíncronos), prevenindo Timeouts quando o cliente requisita compilar PDFs maciços através do Headless Puppeteer. A frente utiliza de React (Inertia.js), combinando painéis responsivos sem expor dezenas de endpoints visíveis.

## Provisionamento Raiz DevOps
Partindo apenas com um acesso SSH da nuvem, assumi toda orquestração Nginx num ambiente Vanilla de Linux Server. Integramos isso nativamente com GitHub Actions CI/CDs automatizando testes pra todo Deploy antes de jogar código para frente sem corromper as sessões dos usuários locais.
