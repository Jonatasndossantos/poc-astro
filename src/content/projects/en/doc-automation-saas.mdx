## End-to-End Development: Document Automation SaaS
The goal was building a '0 to 1' product solving enormous bureaucratic bottlenecks in public administration. This SaaS practically obliterated the time required to compile technical matrices (ETP, DFD) from weeks directly into roughly under 5 minutes without losing legal compliance metrics.

## Engineering & Full Stack
Built using the extremely performant approach combining Laravel 11 with React.js using Inertia.js bridges.
Confronting synchronous bottlenecks, I mapped all extensive processes to background processing using `Laravel Queues and Asynchronous Jobs`.
The engine relied profoundly on strictly engineered JSON Schemas linked through OpenAI's function calling to ban any signs of hallucinations throughout formatting cycles before rendering final Word docs.

## Pure DevOps Configuration
Orchestrated the production environment absolutely from scratch starting with mere raw SSH access. Pre-configured a secure pristine Linux server (Nginx), managed reverse proxy limits, and established rigorous continuous deployments pipelines using GitHub Actions protecting git hooks tightly.
